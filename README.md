## Building

First, install [`mdbook`](https://github.com/rust-lang/mdBook):

```sh
$ cargo install mdbook
```

To build the book, run:

```sh
$ mdbook build
```

To serve the book locally, re-building on any change, run:

```sh
$ mdbook serve
```
